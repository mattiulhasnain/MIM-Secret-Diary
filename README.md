## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/MIM-Secret-Diary.git
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python main.py
   ```
